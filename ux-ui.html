Grundlæggende UI/UX

Tema - O3

Første introduktion til HTML og CSS

I dette tema arbejdede vi med brugeroplevelse og brugergrænseflader. Vi lærte om designprincipper, wireframes, moodboards og planlægning med fokus på brugeren.

Hvad jeg lærte

Mit projekt var Brewphoria – et fiktivt kaffeside-koncept med fokus på æstetik og overskuelighed. Jeg arbejdede med farver, typografi og layout i Figma, før jeg kodede sitet.


Mine udfordringer

Jeg lærte, hvor vigtig balance mellem funktion og følelse er, samt hvordan struktur og grids påvirker helhedsindtrykket. Det gav mig også en bedre forståelse for, hvordan godt design både ser godt ud og fungerer for brugeren.
