Grundlæggende Web

Tema - O2

Første introduktion til HTML og CSS

I tema 2 blev vi introduceret til HTML og CSS og lærte at opbygge en simpel hjemmeside fra bunden. Vi arbejdede med struktur, farver, typografi og opdeling i sektioner som header, main og footer. Jeg lavede et informationssite om computere og testede forskellige farvevalg for at finde mit eget udtryk.

Hvad jeg lærte

Vi arbejdede også med mappestruktur og lærte at bruge programmer som Visual Studio Code og FileZilla til at uploade vores site. Det var lidt overvældende i starten, men fedt at se hvordan tingene begyndte at give mening undervejs.


Mine udfordringer

Det sværeste for mig var at få mit galleri og komponent-side til at fungere med grid – det tog tid at forstå, hvordan kolonner og rækker arbejder sammen. Men det var en lærerig proces, og jeg fik en bedre forståelse for, hvordan man styrer layout og design med CSS.
