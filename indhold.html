Grundlæggende Indhold

Tema - O5

Redesign med fokus på brugervenlighed

I dette tema arbejdede vi i grupper med at redesigne Maude’s Café’s website. Vi startede med research, lavede moodboards og layoutskitser i Figma, og planlagde en løsning med fokus på brugervenlighed og visuelt udtryk.


Samarbejde med Git og GitHub

Jeg havde ansvar for kontakt- og bookingsiden og arbejdede meget med CSS Grid, som var en udfordring for mig. Gennem arbejdet og hjælp fra underviseren fik jeg gradvist styr på det.


Mine udfordringer med Grid

Vi brugte GitHub til versionsstyring og samarbejdede i branches, hvilket gav mig indsigt i, hvordan man undgår at overskrive hinandens kode.


Min udvikling

I dag føler jeg mig faktisk mest sikker i Grid – netop fordi jeg startede usikker og fik lært det hele fra bunden gennem praksis.
